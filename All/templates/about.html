About Data

For this project, a database was obtained from the following link:

https://data.world/theresa-tonev/nyairbnblistings

Data was collected in 2018; therefore, this is the only year stored in the dataset.

Variable Description

Host ID: A unique identifier assigned to each host. This identifier distinguishes one host from another in the dataset.

Host since: The date when the host joined Airbnb or started hosting properties on the platform. It provides information about the tenure of the host.

Neighbourhood: The neighborhood where the host resides. This variable can provide insights into the host's local community.

Latitude: The geographic latitude coordinate of the property. It provides the location information for mapping.

Longitude: The geographic longitude coordinate of the property. It complements the latitude and helps specify the exact location of the property.

Price: The cost per night to rent the property. It represents the pricing of the listed accommodation.

Reviews_per_month: The average number of reviews received by the property per month. It indicates the frequency of guest feedback, reflecting the property's popularity and guest satisfaction.
